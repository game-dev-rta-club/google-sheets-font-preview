# Support

Thanks for using Google Sheets Font Preview.

## Where to get help

Please use GitHub Issues for:

- bug reports
- setup questions
- feature requests
- documentation problems
- Google Sheets / Apps Script behavior that seems confusing

Repository issues:

<https://github.com/game-dev-rta-club/google-sheets-font-preview/issues>

## Before opening an issue

If possible, please include:

- what you expected to happen
- what actually happened
- steps to reproduce the problem
- whether you installed with `clasp`, copied files manually, or used a Marketplace version
- screenshots or a small sample sheet structure if it helps explain the issue

Please do not include private spreadsheet data, secrets, credentials, or proprietary font files.
