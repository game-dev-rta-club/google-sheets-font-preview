# Privacy Policy

Last updated: May 15, 2026

Font Preview Sidebar is a Google Apps Script tool that runs inside Google Sheets. This policy explains what information the tool uses and how it is handled.

## Google user data accessed

Font Preview Sidebar accesses Google user data only from the current Google spreadsheet where the add-on is used. This may include:

- the selected cell and active sheet
- the header row and nearby rows needed for preview
- text cell values used for base-language and localization previews
- screenshot column image data or image display data
- note, width, and height column values
- spreadsheet metadata needed to display and update the sidebar
- project options saved for the current spreadsheet

The tool stores project options in the spreadsheet's Apps Script document properties. These options may include visible panel settings, default language settings, preview size settings, user-defined non-language column names, and custom fixed header names.

## Google user data usage

Font Preview Sidebar uses Google Sheets data only to provide its sidebar features. Specifically, the tool uses spreadsheet data to:

- preview spreadsheet text with selected fonts
- show screenshot and note context from the same row
- display size-aware text areas based on width and height values
- switch preview languages based on spreadsheet headers
- update cells when you edit text from the sidebar
- create a sample sheet when requested from the Help screen
- save and load project options for the current spreadsheet

The tool does not use Google user data for advertising, analytics, tracking, sale of data, or AI/ML model training.

## Google user data storage and sharing

Font Preview Sidebar does not send spreadsheet data to an external server controlled by this project. Spreadsheet data remains within Google Sheets and Google Apps Script infrastructure.

Project options are stored only in the current spreadsheet's Apps Script document properties. Local font files are used in your browser session for preview purposes and are not uploaded to an external server controlled by this project.

The tool does not share Google user data with third parties except as necessary for Google Sheets and Google Apps Script to provide the service.

## Limited use

Font Preview Sidebar's use and transfer of information received from Google APIs adheres to the [Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy), including the Limited Use requirements.

## Local fonts

When you drag and drop a local font file into the sidebar, the font is loaded in your browser for preview purposes. Local font files are used in the browser session and are not uploaded to an external server controlled by this project.

## Data sharing and third-party services

This project does not include analytics, advertising, tracking, or a separate backend service.

The tool runs on Google's Apps Script and Google Sheets infrastructure, so use of the tool is also subject to Google's terms and privacy policies.

## Project website

The public project website is hosted with GitHub Pages. Visiting the website may share normal browser request information with GitHub.

## External links

The sidebar and documentation may link to the GitHub repository or related documentation. Opening those links may share normal browser request information with the destination site.

## Security

Please do not put secrets, passwords, private keys, or other highly sensitive credentials in spreadsheets used with this tool.

If you believe you found a security or privacy issue, please follow the process in the [Security Policy](https://github.com/game-dev-rta-club/google-sheets-font-preview/blob/main/SECURITY.md).

## Contact

For questions about this policy, please open an issue in the [GitHub repository](https://github.com/game-dev-rta-club/google-sheets-font-preview/issues).
