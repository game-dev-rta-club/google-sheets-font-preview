# Marketplace assets

This folder contains draft assets for a future Google Workspace Marketplace listing.

## Current assets

- `icon.svg`: editable source icon
- `icon-32.png`: small Marketplace icon candidate
- `icon-128.png`: large Marketplace icon candidate
- `card-banner-220x140.png`: card banner candidate

## Screenshot candidates

The README images in `docs/images/` are useful product screenshots, but Marketplace screenshots should be exported in an accepted Marketplace size before submission.

Recommended submission target:

- `1280x800`

Other accepted sizes may include:

- `640x400`
- `2560x1600`

## Notes

- Keep images free of private spreadsheet data.
- Prefer screenshots that show the sidebar next to a realistic sheet.
- Update screenshots whenever the UI changes significantly.
